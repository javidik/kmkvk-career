6. БЫСТРЫЙ СТАРТ НА GITHUB PAGES
═══════════════════════════════════════════════════════════════════════

1️⃣  Скачайте ВСЕ файлы (они уже созданы выше)

2️⃣  Создайте структуру папок:
    mkdir -p kmkvk-career-site/{css,js,assets}

3️⃣  Разместите файлы в соответствующих папках

4️⃣  Создайте GitHub репозиторий:
    https://github.com/new

5️⃣  Загрузите файлы на GitHub
    (через веб-интерфейс или git commands)

6️⃣  Включите GitHub Pages в Settings → Pages

7️⃣  Сайт будет доступен по адресу:
    https://ВАШ-ЛОГИН.github.io/kmkvk-career-site/
